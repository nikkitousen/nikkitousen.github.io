Location Guard

Location Guard is a browser extension that allows to protect your location while using location-aware websites, by adding controlled noise to it. It supports Chrome / Chromium, Mozilla Firefox / Firefox for Android and Opera.

This extension is based on the Planar Laplace Mechanism presented and studied in my PhD thesis and in this paper. The tool is maintained by Kostas Chatzikokolakis and Marco Stronati.
